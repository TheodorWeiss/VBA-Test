Да, задача понятна.

И тут действительно логика для M и N расходится:
	•	N (по J) — считаем Werbefenster, где есть хотя бы один J=1
	•	M (по I) — считаем Werbefenster, где:
	•	для одиночных B=1 или B=2: просто I=1
	•	для дробных B<1:
есть хотя бы один I=1
и нет ни одного J=1
То есть найденные артикли этого Werbefenster лежат только в H/I, и хотя бы один попал именно в I.

Это важное уточнение, иначе окна “только H” начали бы ошибочно попадать и в M.

⸻

1. Логика ещё раз коротко

M

Если B=1 или B=2
	•	M = 1, если I=1
	•	иначе 0

Если B<1

Считаем только по первой строке группы:
	•	если в группе есть хотя бы один I=1
	•	и в группе нет ни одного J=1
→ M = 1
	•	иначе 0

⸻

N

Если B=1 или B=2
	•	N = 1, если J=1
	•	иначе 0

Если B<1

Считаем только по первой строке группы:
	•	если в группе есть хотя бы один J=1
→ N = 1
	•	иначе 0

⸻

2. Что использовать как “первая строка группы”

Тут ничего нового не нужно:
можно использовать уже существующую колонку K.

Если в K у тебя уже стоит логика:
	•	для B=1 и B=2 → 1
	•	для дробных → HasSimilarAbove(...)=FALSE

то этого достаточно и для M/N.

⸻

3. Нужна ещё одна UDF

Текущая AllSimilarHaveH1(...) для M/N уже не подходит.
Нужна более универсальная функция:

AnySimilarHaveFlag1(...)

Она будет отвечать:

есть ли в этой группе хотя бы один артикль, у которого данный флаг = 1

⸻

VBA-код

В тот же стандартный модуль, где уже лежат HasSimilarAbove и AllSimilarHaveH1, добавь:

Public Function AnySimilarHaveFlag1( _
    ByVal curArtCell As Range, _
    ByVal curBCell As Range, _
    ByVal artRange As Range, _
    ByVal bRange As Range, _
    ByVal flagRange As Range) As Boolean

    Dim i As Long
    Dim sameGroup As Boolean

    If artRange.Rows.Count <> bRange.Rows.Count _
       Or artRange.Rows.Count <> flagRange.Rows.Count Then
        AnySimilarHaveFlag1 = False
        Exit Function
    End If

    For i = 1 To artRange.Rows.Count
        sameGroup = False

        If SameB(curBCell.Value, bRange.Cells(i, 1).Value) Then
            If artRange.Cells(i, 1).Row = curArtCell.Row Then
                sameGroup = True
            ElseIf IsSimilarArticle(curArtCell.Value, artRange.Cells(i, 1).Value) Then
                sameGroup = True
            End If
        End If

        If sameGroup Then
            If flagRange.Cells(i, 1).Value = 1 Then
                AnySimilarHaveFlag1 = True
                Exit Function
            End If
        End If
    Next i

    AnySimilarHaveFlag1 = False
End Function

4. Формула для N

Допустим, блок сети — строки 143:170.
Тогда в N143:

=WENN(ODER($B143=1;$B143=2);
     WENN($J143=1;1;0);
     WENN($K143=1;
          WENN(AnySimilarHaveFlag1($A143;$B143;$A$143:$A$170;$B$143:$B$170;$J$143:$J$170);1;0);
          0))

Протянуть вниз до N170.

⸻

5. Формула для M

В M143:

=WENN(ODER($B143=1;$B143=2);
     WENN($I143=1;1;0);
     WENN($K143=1;
          WENN(UND(
                   AnySimilarHaveFlag1($A143;$B143;$A$143:$A$170;$B$143:$B$170;$I$143:$I$170);
                   NICHT(AnySimilarHaveFlag1($A143;$B143;$A$143:$A$170;$B$143:$B$170;$J$143:$J$170))
                  );
               1;
               0);
          0))

Протянуть вниз до M170.

⸻

6. Что именно делает формула M

Для дробных групп она проверяет сразу два условия:
	1.	AnySimilarHaveFlag1(...I...)
→ есть ли в группе хотя бы один I=1
	2.	NICHT(AnySimilarHaveFlag1(...J...))
→ нет ли в группе J=1

Только если оба условия выполняются, группа получает 1.

⸻

7. Почему это правильно

Так ты получаешь красивое разбиение:
	•	L — чисто H-логика
	•	M — I-логика, но только если группа не уходит в J
	•	N — всё, где есть J

И для дробных Werbefenster это уже выглядит логически согласованно.

⸻

8. Важное допущение, которое я сейчас использую

Я исхожу из того, что для M нужно:
	•	не просто отсутствие J
	•	а именно наличие хотя бы одного I

То есть группа:
	•	H=1, H=1, J=0
не должна попадать в M.

Если же ты вдруг хотел, чтобы такие “чисто H”-группы тоже шли в M, это надо менять. Но по твоему описанию это было бы странно и дало бы двойной счёт с L.

⸻

9. Если захочешь сделать понятнее и быстрее для глаз

Можно использовать O и P как вспомогательные:
	•	O = есть ли в группе хотя бы один I=1
	•	P = есть ли в группе хотя бы один J=1

Тогда:

O143

=WENN($K143=1;AnySimilarHaveFlag1($A143;$B143;$A$143:$A$170;$B$143:$B$170;$I$143:$I$170);0)

P143

=WENN($K143=1;AnySimilarHaveFlag1($A143;$B143;$A$143:$A$170;$B$143:$B$170;$J$143:$J$170);0)

И тогда итоговые формулы будут короче:

M143

=WENN(ODER($B143=1;$B143=2);
     WENN($I143=1;1;0);
     WENN($K143=1;WENN(UND($O143=1;$P143=0);1;0);0))

N143

=WENN(ODER($B143=1;$B143=2);
     WENN($J143=1;1;0);
     WENN($K143=1;WENN($P143=1;1;0);0))
